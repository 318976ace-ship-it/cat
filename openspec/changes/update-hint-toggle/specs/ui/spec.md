## ADDED Requirements
### Requirement: 提示顯示切換
系統 MUST 提供按鈕可顯示/隱藏螢幕上的提示與鍵位說明，讓手機玩家能擴大鋼琴遊戲區。

#### Scenario: 手機隱藏提示以放大遊戲區
- **GIVEN** 玩家位於小螢幕（手機或狹窄視窗）
- **WHEN** 玩家點擊提示切換將提示/鍵位說明隱藏
- **THEN** 提示/鍵位說明區塊被收起並自可視區消失
- **AND** 鋼琴/遊戲區域隨之放大以使用騰出的空間
- **AND** 隱藏狀態在同一瀏覽器重新整理後仍被記住。

#### Scenario: 需要時重新顯示提示
- **GIVEN** 提示/鍵位說明目前已隱藏
- **WHEN** 玩家點擊提示切換以顯示提示
- **THEN** 提示/鍵位說明重新顯示，版面可讀且不與琴鍵重疊
- **AND** 顯示狀態會被保存，供後續回到頁面時使用。
